## Suggested command mapping (CI-ish defaults)
- Cargo:
  - cargo fmt --check
  - cargo clippy -- -D warnings
  - cargo test
- Go:
  - gofmt check (recommended: sh -c 'test -z "$(gofmt -l .)"')
  - go vet ./...
  - go test ./...
- Elixir:
  - mix format --check-formatted
  - mix compile --warnings-as-errors
  - mix test
- Python:
  - if ruff on PATH: ruff format --check . ; ruff check . ; pytest
  - else if black on PATH: black --check . ; (optional) flake8 ; pytest
  - else bail with actionable message
- Node:
  - <pm> run lint
  - <pm> test
  - (optional) <pm> run format:check if you want
- Gradle:
  - ./gradlew check (wrapper) or gradle check
- Maven:
  - mvn test
- Ruby:
  - bundle exec rake test
- Meson:
  - meson test -C builddir
- CMake:
  - ctest --test-dir build
- Make:
  - make test

If you think some ecosystems should bail instead of guessing, document the decision in README.
