## What to build

Implement Elixir, Java, and Ruby language adapters. Follow the pattern established by existing adapters in `crates/manifest/src/adapters/lang/`.

### Files to create

1. **crates/manifest/src/adapters/lang/elixir.rs** — Elixir adapter:
   - `pub struct ElixirAdapter;`
   - `detect`: true when `ctx.kind == ProjectKind::Elixir { .. }`
   - `extract` via tree-sitter-elixir:
     - `defmodule X do` → TypeDef (kind: Class, since Elixir modules are the primary organizational unit)
     - `def func` → Function (Public)
     - `defp func` → Function (Private)
     - `@behaviour X` / `@impl X` → implements: ["X"]
     - `@moduledoc`, `@doc` → skip (doc attributes)
     - `use X`, `import X`, `alias X` → noted in module imports
   - Filter: `.ex` and `.exs` files, skip `_build/`, `deps/`
   - Priority: 100, Layer: Language

2. **crates/manifest/src/adapters/lang/java.rs** — Java adapter:
   - `pub struct JavaAdapter;`
   - `detect`: true when `ctx.kind == ProjectKind::Gradle { .. } || ctx.kind == ProjectKind::Maven`
   - `extract` via tree-sitter-java:
     - `public class X` → TypeDef (kind: Class)
     - `public interface X` → TypeDef (kind: Interface)
     - `public enum X` → TypeDef (kind: Enum)
     - `class X implements Y, Z` → implements: ["Y", "Z"]
     - `class X extends Base` → implements: ["Base"]
     - `public` methods → Function (Public), others Private
     - `@Override`, `@Test` and other annotations → noted
     - Package structure → Module
   - Filter: `.java` files, skip `build/`, `target/`, `.gradle/`
   - Priority: 100, Layer: Language

3. **crates/manifest/src/adapters/lang/ruby.rs** — Ruby adapter:
   - `pub struct RubyAdapter;`
   - `detect`: true when `ctx.kind == ProjectKind::Ruby`
   - `extract` via tree-sitter-ruby:
     - `class X` → TypeDef (kind: Class)
     - `class X < Base` → implements: ["Base"]
     - `module X` → TypeDef (kind: Trait) or Module depending on usage
     - `def method` → Function (Public by default)
     - `attr_accessor :x`, `attr_reader :x`, `attr_writer :x` → fields on the class
     - After `private` keyword, methods become Private visibility
   - Filter: `.rb` files, skip `vendor/`, `.bundle/`
   - Priority: 100, Layer: Language
