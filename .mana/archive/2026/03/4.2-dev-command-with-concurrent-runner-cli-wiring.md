## Task
Add `uu dev` command that runs dev scripts concurrently in workspaces, with colored prefixed output. Wire it into the CLI.

## Steps

### 1. Add dependency to `crates/uu/Cargo.toml`
- Add `ctrlc = { version = "3", features = ["termination"] }`

### 2. Create `crates/uu/src/cmd/dev.rs`

Three modes:
- **Workspace + package args**: `uu dev api web` -> run those packages dev scripts
- **Workspace + no args**: `uu dev` -> run ALL packages dev scripts
- **Non-workspace**: `uu dev` -> run `{manager} run dev` in current dir

```rust
//! `uu dev` — start dev servers, workspace-aware.

use std::env;
use std::io::{BufRead, BufReader};
use std::process::{Command, Stdio};
use std::sync::atomic::{AtomicBool, Ordering};
use std::sync::Arc;
use std::thread;

use anyhow::{bail, Result};
use uu_detect::{detect_node_workspace, NodePM, ProjectKind, WorkspacePackage};

use crate::runner::{self, step, Step};
```

**`execute(dry_run, packages, extra_args)`**:
- Call `runner::detect_project()`
- If Node.js, call `detect_node_workspace(&dir)`
- If workspace found, filter packages (or use all), call `run_concurrent()`
- If no workspace or non-Node, fall back to `single_dev()` which runs `{manager} run dev`
- For non-Node ecosystems (Cargo, Go, etc.), `single_dev` does the same as `uu run`

**`run_concurrent(kind, packages, manager, dry_run)`**:
- Print detected line using `runner::style()`: `  detected Node.js workspace (pnpm, N packages)`
- If dry_run, print each package with `would run` and return
- Validate all requested package names exist (bail with error listing available packages if not)
- Compute max name length for aligned output
- Set up `Arc<AtomicBool>` for shutdown flag
- Register `ctrlc::set_handler` that sets the flag and prints a shutdown message
- For each package, spawn child process:
  - `Command::new(manager_cmd).args(["run", "dev"]).current_dir(&pkg.path).stdout(Stdio::piped()).stderr(Stdio::piped())`
  - Print `running {name} · {dev_script}` line
- For each child, spawn 2 threads (stdout + stderr reader):
  - `BufReader::new(stream).lines()` loop
  - Print with colored prefix: `  [{name:>width$}] {line}`
  - Color rotation: ["36", "33", "35", "32", "34"] (cyan, yellow, magenta, green, blue)
- Main thread: loop checking shutdown flag with `thread::sleep(100ms)`
- On shutdown: kill all children, join all threads

### 3. Register module in `crates/uu/src/cmd/mod.rs`
Add line: `pub(crate) mod dev;`

### 4. Wire up CLI in `crates/uu/src/main.rs`

Add to the Commands enum:
```rust
/// Start dev servers (workspace-aware — runs packages concurrently)
Dev(DevArgs),
```

Add DevArgs struct (NOT inside ProjectArgs — dev has different args):
```rust
#[derive(Args)]
struct DevArgs {
    /// Show what would run without executing
    #[arg(short = 'n', long)]
    dry_run: bool,

    /// Workspace packages to run (omit for all)
    packages: Vec<String>,
}
```

Add match arm in main():
```rust
Commands::Dev(a) => cmd::dev::execute(a.dry_run, a.packages, vec![]),
```
