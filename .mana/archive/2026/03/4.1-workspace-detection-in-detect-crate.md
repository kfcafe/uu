## Task
Add workspace detection to the `uu-detect` crate. This provides the data layer that `uu dev` will use.

## Steps

1. Add `serde` and `serde_json` as dependencies to `crates/detect/Cargo.toml`

2. Add these public types and function to `crates/detect/src/lib.rs`:

```rust
/// A package in a workspace that has a "dev" script.
#[derive(Debug, Clone)]
pub struct WorkspacePackage {
    pub name: String,       // directory name, e.g. "api", "web"
    pub path: PathBuf,      // absolute path to the package directory
    pub dev_script: String, // value of the "dev" script from package.json
}

/// Detect Node.js workspace packages that have a "dev" script.
///
/// Checks for pnpm-workspace.yaml first, then "workspaces" field in package.json.
/// Returns None if not a workspace. Returns Some(vec![]) if workspace but no dev scripts.
pub fn detect_node_workspace(dir: &Path) -> Option<Vec<WorkspacePackage>>
```

3. Implementation details for `detect_node_workspace`:
   - **pnpm**: Read `pnpm-workspace.yaml` manually (no YAML crate). Format is simple: lines with `- "pattern"` or `- pattern`. Extract glob patterns. Lines starting with `!` or `- "!` are exclusions — skip those patterns.
   - **npm/yarn/bun**: Read `package.json`, parse with serde_json, check for `"workspaces"` array.
   - For each glob pattern (e.g. `packages/*`), use `std::fs::read_dir` to list matching directories. Only handle simple `prefix/*` patterns (split on `/*`, read_dir the prefix).
   - For each directory found, read its `package.json`, look for `scripts.dev`. If present, create a `WorkspacePackage`.
   - Sort results by name for deterministic output.

4. Add tests (all test names must contain "workspace" so the verify filter catches them):
   - `detect_pnpm_workspace` — tempdir with pnpm-workspace.yaml + 2 packages (one with dev, one without)
   - `detect_npm_workspace` — tempdir with package.json containing "workspaces" + packages
   - `no_workspace_returns_none` — plain package.json, no workspace config
   - `workspace_no_dev_scripts` — workspace exists but no dev scripts -> Some(vec![])
