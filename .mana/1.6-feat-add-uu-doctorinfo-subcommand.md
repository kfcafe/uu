## Task
Add a new subcommand: `uu doctor` (and alias `uu info`).

This should help users understand what `uu` detected and what external tools are needed.

## Desired behavior
- If a project is detected, print the same "detected <kind> (<file>)" header as other commands.
- Show a short report of required external programs for that ecosystem (and whether they are on PATH).
  - Example for Cargo: cargo, rustfmt, clippy
  - Example for Node: chosen package manager (npm/yarn/pnpm/bun)
  - Example for Ports: lsof
- Exit code should be 0 even if some tools are missing; it is an informational command.
- If no project is detected, show the supported project file table (uu_detect::supported_table()) and exit 0.
