## What to build

Implement Swift, C#, and C/C++ language adapters. Follow the pattern established by existing adapters in `crates/manifest/src/adapters/lang/`.

### Files to create

1. **crates/manifest/src/adapters/lang/swift.rs** — Swift adapter:
   - `pub struct SwiftAdapter;`
   - `detect`: true when `ctx.kind == ProjectKind::Swift`
   - `extract` via tree-sitter-swift:
     - `struct X` → TypeDef (kind: Struct)
     - `class X` → TypeDef (kind: Class)
     - `protocol X` → TypeDef (kind: Protocol)
     - `enum X` → TypeDef (kind: Enum)
     - `public func` → Function (Public)
     - `func` (default internal) → Function (Internal visibility)
     - `private func` → Function (Private)
   - Filter: `.swift` files, skip `.build/`
   - Priority: 100, Layer: Language

2. **crates/manifest/src/adapters/lang/csharp.rs** — C# adapter:
   - `pub struct CSharpAdapter;`
   - `detect`: true when `ctx.kind == ProjectKind::DotNet { .. }`
   - `extract` via tree-sitter-c-sharp:
     - `public class X` → TypeDef (kind: Class)
     - `public interface X` → TypeDef (kind: Interface)
     - `public enum X` → TypeDef (kind: Enum)
     - `public record X` → TypeDef (kind: Struct)
     - `public` methods → Function (Public)
     - `internal` → Function (Internal)
     - Class implements/extends → implements list
     - Namespace → Module
   - Filter: `.cs` files, skip `bin/`, `obj/`
   - Priority: 100, Layer: Language

3. **crates/manifest/src/adapters/lang/c_cpp.rs** — C/C++ adapter:
   - `pub struct CCppAdapter;`
   - `detect`: true when `ctx.kind` is `CMake`, `Meson`, or `Make`
   - `extract` via tree-sitter-c and tree-sitter-cpp:
     - Use tree-sitter-c for `.c` and `.h` files
     - Use tree-sitter-cpp for `.cpp`, `.cc`, `.cxx`, `.hpp`, `.hxx` files
     - `struct X` → TypeDef (kind: Struct)
     - `typedef` → TypeDef (kind: TypeAlias)
     - `enum X` → TypeDef (kind: Enum)
     - `class X` (C++) → TypeDef (kind: Class)
     - Function declarations in headers → Function (Public)
     - `static` functions → Function (Private)
     - `union X` → TypeDef (kind: Union)
   - Focus on header files for public API surface
   - Filter: `.h`, `.hpp`, `.c`, `.cpp`, `.cc`, `.cxx`, `.hxx` files
   - Priority: 100, Layer: Language
