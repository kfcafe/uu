## Task
Add a new project-aware subcommand: `uu check` (aka fast typecheck/build).

This should be a faster feedback loop than `uu test`: compile/typecheck without running tests where possible.

## CLI shape
- Use the existing ProjectArgs style
  - -n/--dry-run
  - passthrough args after `--` (append to the LAST step)
- Consider adding a Clap alias `typecheck`.

## Implementation notes

- Wire into main + cmd/mod.rs

## Suggested command mapping
- Cargo: cargo check
- Go: go test -run=^$ ./...
- Elixir: mix compile --warnings-as-errors
- Python: python -m compileall -q . (OK to bail instead if you think side effects are too surprising)
- Node: <pm> run typecheck
- Gradle: ./gradlew build -x test (wrapper) or gradle build -x test
- Maven: mvn -DskipTests package
- Meson: meson compile -C builddir
- CMake: cmake -B build then cmake --build build (or just build if you want)
- Make: make

For ecosystems where the default is questionable, bail with a helpful message.
