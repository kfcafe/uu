## Task
Improve `crates/uu/src/cmd/run.rs` based on vibecheck findings.

## Findings (top 20)
- [low] edge-cases — `crates/uu/src/cmd/run.rs:24-27`
  - Python with `uv: true` runs `uv run manage.py runserver` / `uv run app.py` etc. But `uv run` expects a module or script — for Django, the correct invocation would be `uv run python manage.py runserver`. Without `python` prefix, `uv run manage.py` tries to find `manage.py` as a command in the venv, which may not work.
  - Suggested fix: Use `uv run python manage.py runserver` (pass `["run", "python"]` as prefix_args) instead of `uv run manage.py runserver`.
